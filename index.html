HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML Tables
<thead> Table Head The set rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The container within the row that contains individual data
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columbs a td element should span.
rowspan Row Span Defines how many rows a td element should span

CSS Table Reference

Table Layout

Code What it does
table-layout: auto; The widths of the table and its cells are adjusted to fit the content.
table-layout: fixed; The width of the table is fixed and needs to be specified explicitly

Table Borders

Code What it does
table { border: 1px solid;} Gives a border to the entire table element
td, th { border: 1px solid;} Gives a border to the th and td elements
table { border-collapse: collapse;} Borders on all of the table elements creates a double line effect; This collapses them into one.
th, td { border-bottom: 1px solid;} Gives just a bottom border to each of the elements

Table Effects

Code What it does
tr:hover { background-color: magenta;} Gives the table row a different background color when you hover over it.
tr:nth-child(even) { background-color: aqua;} Alternates the background color for the table rows, creating stripes
